# Ficha 1 — Exercícios introdutórios (Threads)

Esta ficha aborda conceitos básicos de programação concorrente em Java,
nomeadamente criação e controlo de threads, pausas (`sleep`) e interrupção.

## Exercícios
- **Ex1** — Simulação de processamento de pedidos em paralelo.
- **Ex2** — Monitorização periódica de temperatura.
- **Ex3** — Simulação de notificações/mensagens concorrentes.
- **Ex4** — Leitura e monitorização de logs.

Cada exercício encontra-se numa subpasta (`ex1` a `ex4`) com o respetivo código
e README explicativo.
